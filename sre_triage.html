How It Works

SRE Triage is an OpenEnv-compliant benchmark for training and evaluating AI agents on production incident response. Drop into a simulated outage, investigate using logs/metrics/alerts, and identify the root cause before running out of steps.

TIP
Start with list_alerts every episode. Then query logs on the alerting service, confirm with metrics, run the appropriate playbook or rollback, and finally submit_diagnosis. Don't waste steps on services that aren't alerting.

Scoring Formula
Total Score = 0.50 × Correctness + 0.30 × Efficiency + 0.20 × Speed
Correctness = 1.0 if root_cause + service both correct, else 0.0
Efficiency = max(0, 1 - (extra_steps / max_steps))
Speed = max(0, 1 - (elapsed_seconds / time_limit))
